情報技術の歴史（時系列要約）

古代～印刷技術の発明

有史以前、人々はのろしやジェスチャなどの記号的な表現で情報を伝達していました 。やがて音声言語や文字言語が発達し、情報を書き記して記録・保存できるようになります 。紀元前には紙が発明され、手紙や本という形でより多くの情報を持ち運べるようになりました 。さらに印刷技術の発明により、同じ内容を多くの人に伝えることが容易になりました 。

1820年代～1890年代：電気通信と計算機の黎明

  • 1820～30年代: 数学や物理の計算を自動化する機械式計算機が発明されました 。代表例として日本のタイガー計算機（初代機は1923年発売）が挙げられます 。
  • 1830年代: 電気通信が発明され、離れた場所へ情報を伝えられるようになりました 。「0と1」によるデジタル通信の原型であるモールス信号もこの時期に登場します 。
  • 1870年代: グラハム・ベルとエリシャ・グレイがほぼ同時に電話を発明し、音声を電気信号として遠隔地に伝えることが可能になりました 。
  • 1890年代:
    • グリエルモ・マルコーニが無線でモールス信号を送ることに成功しました 。
    • 米国でパンチカードを用いた統計処理システムが採用され、このシステムを開発した会社は後にIBM社（1924年に社名変更）となります 。

1920年代～1940年代：電子計算機の誕生と情報理論

  • 1920～30年代: ブラウン管の発明などを経て、テレビ放送が始まりました 。
  • 1930年代: クロード・シャノンが、あらゆる情報を0と1の「ビット」で表現し、ブール代数で論理演算を行う情報理論の基礎を築きました 。これは、後のデジタル技術の根幹をなす理論です 。
  • 1940年代:
    • 1943年: アラン・チューリングらが参加し、ドイツの暗号を解読するためのコロッサスが英国で極秘開発されました。
    • 1945年: ヴァネヴァー・ブッシュが、現在のWWWの原型となるハイパーテキストの思想を持つMemexの構想を発表しました。
    • 1946年: 米国で世界初のプログラム内蔵方式コンピュータであるENIACが実用化されました 。開発にはジョン・フォン・ノイマンも参加し、この設計はノイマン型アーキテクチャとして現在のコンピュータの主流となっています。

1950年代～1960年代：メインフレームと新たなコンピュータの概念

  • 1950～60年代: IBMのSystem/360 やDEC社のPDPシリーズ などに代表されるメインフレーム（大型汎用計算機）がビジネスシーンの主流となりました 。これらは、データ処理を中央のコンピュータで行う集中コンピューティングという形態でした 。
  • 1958年: 米国テキサス・インスツルメンツ社が**IC（集積回路）**を発明し、半導体チップ上に電子回路を構成することが可能になりました 。
  • 1960年代:
    • テッド・ネルソンが、文字や画像がリンクで繋がるハイパーテキストの概念を持つXanaduを構想しました 。
    • ダグラス・エンゲルバートがマウスを発明し、オンラインで文書を編集するNLSを開発しました 。
    • アイヴァン・サザランドが、対話的なコンピュータグラフィックスの原点であるSketchpad や、ヘッドマウントディスプレイ（HMD）の原型を開発しました 。
    • アラン・ケイがパーソナルコンピュータの構想を提唱しました 。

1970年代以降：マイクロプロセッサとパーソナルコンピュータの実現

  • 1971年:
    • 世界初の1チップマイコンTMS1000が開発されました 。
    • 日本のビジコン社の依頼で、米国インテル社が世界初のマイクロプロセッサ（CPU）である4004を開発しました 。この後、半導体技術の発展と共にコンピュータは飛躍的に進化します。
  • 1973年: アラン・ケイらが、WIMPインタフェース（ウィンドウ、アイコン、メニュー、ポインタ）を実装したXerox Altoを開発しました。
  • 1977年: アラン・ケイが、現在のノートPCやタブレットに繋がるDynabook構想を発表しました。
  • 1980年代～: これまでの集中コンピューティングから、個々のコンピュータが処理を分担する分散コンピューティングへと移行が進みました 。現代のプロセッサは、一つのチップに複数のCPUコアを搭載するマルチコアや、CPU以外の機能も統合したSoC（System on Chip）へと発展しています。
